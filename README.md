# Myers_Johnny_JSON-Based_Learning_Chatbot

Class:

COP2800-3 held by Professor Stevan Stankovic.


Prompt:
Write a program that implements a console-based chatbot that reads predefined question–answer pairs from a JSON file and uses them to communicate with a user.
The program starts by loading a JSON data file into a data structure of your choice using file I/O.
The chatbot must continuously prompt the user for input in an infinite loop, process input in a case-insensitive manner, and attempt to match it to a corresponding key in the dataset. If a match is not found, the program prompt a user for an answer in order "to learn".
The program must allow the user to terminate the interaction by entering a specific command (i.e., “exit” or "goodbye").
The overall implementation should demonstrate proper use of classes, methods, control structures, and basic object-oriented design principles.
This program shows how structured data processing can resemble an AI-like behavior and how chatbots rely on pattern matching rather than true understanding.

Grading:

Basic Java functionality                     30 points

JSON parsing and file I/O                    20 points

Program design and OOP                       15 points

Exception handling                           10 points

Partial matching/multi responses             10 points

Reflection                                   15 points
